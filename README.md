# Realtime Chat App

A simple realtime chat app built with **Next.js (TypeScript)** and **Ably Realtime**. Multiple users can send and receive messages instantly.

**Deployed App:** [https://chat-81ziib8cv-rjs-projects-2b8d78ca.vercel.app/](https://chat-81ziib8cv-rjs-projects-2b8d78ca.vercel.app/)  
**GitHub Repo:** <your-repo-url>

## Setup

1. Clone the repo and install dependencies:

```bash
git clone <your-repo-url>
cd chat-app
npm install
````

2. Add your Ably key in `.env.local`:

```env
NEXT_PUBLIC_ABLY_API_KEY=your-client-side-ably-key
```

3. Run locally:

```bash
npm run dev
```
